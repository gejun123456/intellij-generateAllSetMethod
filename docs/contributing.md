# Contributing

We welcome contributions to the GenerateAllSetter plugin!

## Getting Started

1. **Import the project** into IntelliJ IDEA
2. **Run the `runIde` Gradle task** to launch a test IDE with the plugin
3. **Change the code** and run the task again to see your changes

## Development Setup

The project uses Gradle as its build system. To get started:

```bash
git clone https://github.com/gejun123456/intellij-generateAllSetMethod.git
cd intellij-generateAllSetMethod
./gradlew runIde
```

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Reporting Issues

If you find a bug or have a feature request, please [open an issue](https://github.com/gejun123456/intellij-generateAllSetMethod/issues) on GitHub.
