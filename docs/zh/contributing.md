# 贡献指南

我们欢迎对 GenerateAllSetter 插件的贡献！

## 开发环境设置

1. **将项目导入** IntelliJ IDEA
2. **运行 `runIde` Gradle 任务** 启动一个包含插件的测试 IDE
3. **修改代码** 然后再次运行任务查看效果

## 开发准备

本项目使用 Gradle 作为构建系统。开始开发：

```bash
git clone https://github.com/gejun123456/intellij-generateAllSetMethod.git
cd intellij-generateAllSetMethod
./gradlew runIde
```

## 提交 Pull Request

1. Fork 本仓库
2. 创建一个功能分支
3. 进行你的修改
4. 提交 Pull Request

## 报告问题
